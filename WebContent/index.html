3SMK PROMOTE

(주)3SMK는 세계 최고 수준의 나노 금형 및 홀로그램 보안 패턴을 전문문으로 제작하는 업체로 한국전기연구원과는 마이크로 나노 장비 공공 개발 사업을 진행하고 한국조폐공사와 홀로그램 보안 패턴 개발 사업을 공동 수행하는 회사입니다. 또한 국책기술지원과제인 '홀로그램을 이용한 차세대 보안 패턴 개발' 주관기업으로 선정되어 과제를 진행 중에 있으며 이를 기반으로 각 기업체에 홀로그램 보안 패턴을 제공하고 있습니다.

3SMK is a company that specializes in world-class nano mold and holographic security patterns. It is a company that conducts public development of micro nanotechnology equipment with the Korea Electrotechnical Institute and co-hosts the holographic security pattern development project with the Korea Minting and Printing Corporation. In addition, the company has been selected as the host company of the state-run technical support project 'Next Generation Security Pattern Development Using Hologram' and is currently providing holographic security patterns to each company based on this project.
